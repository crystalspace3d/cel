2.4.1 Unix/Linux

This section describes how to build Crystal Entity Layer on Unix style platforms such as Linux, FreeBSD, Solaris, etc.

Crystal Entity Layer now uses jam as it's preferred build system. The GNU make build tool is no longer used.

In order to build Crystal Entity Layer on this platform:

  1. Ensure that Crystal Space has been compiled correctly.

  2. Ensure that the CRYSTAL environment variable is set correctly. For example:
    export CRYSTAL=~/CS

  3. From the cel directory run the configure script, with the command:
    ./configure

  4. Use jam to build the project
    jam

In order to test if Crystal Entity Layer has compiled correctly run the celtst test application.
