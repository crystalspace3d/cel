2.4.4 Windows using Cygwin

These instructions detail how to build Crystal Entity Layer under Cygwin. They assume that you already have Cygwin installed and operating. They also assume that you have the Jam build environment running under Cygwin.

  1. Ensure that Crystal Space has been compiled correctly.

  2. Ensure that the CRYSTAL environment variable is set correctly. For example:
    export CRYSTAL=~/CS

  3. From the cel directory run the configure script, with the command:
    ./configure

  4. Use jam to build the project
    jam
