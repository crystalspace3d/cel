3.1 Overview

The applications currently bundled with Crystal Entity Layer are:

celstart

Celstart is a simple application that can be used to fire up a CEL game. As a parameter it expects a simple config file or zip file containing an appropiate config file which fully describes how the game should start and what parameters it expects.

It can be used as ‘runtime’ for working fully from scripting without need of any c++ coding.

celtest

This is a test application designed to test the various Crystal Entity Layer modules and give a basic demonstration of what some of the property classes do.

bootstrap

Bootstrap is an older application that is also able to run maps fully consisting of xml and scripting, but is less versatile than celstart and now partially deprecated.
