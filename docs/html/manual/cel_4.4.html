4.4 Addons

Although you can fully work with Crystal Entity Layer by using C++ API calls, Crystal Entity Layer has a lot of support for XML. To supports this Crystal Entity Layer uses the addon system that is implemented in the Crystal Space map loader. Basically the addon system allows one to write plugins that plug into the loader and allow someone to add additional XML parts to the standard world file. Crystal Entity Layer defines several addons.
