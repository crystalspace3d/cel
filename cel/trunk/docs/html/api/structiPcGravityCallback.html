Detailed Description

Inherit this class if you want to know when gravity is applied to a certain iPcLinearMovement.

This is usefully if you for example want to apply certain gravity or something on the way up, but not on the way down (Parachutes and things like that). Register this callback with AddGravityCallback in iPcLinearMovement, and remove with RemoveGravityCallback
