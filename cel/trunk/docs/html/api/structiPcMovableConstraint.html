Check if an object can move.

Returns:

  • CEL_MOVE_FAIL: if no movement is possible.
  • CEL_MOVE_SUCCEED: if movement is possible.
  • CEL_MOVE_PARTIAL: if object can move partially.
'pos' will contain the final position that this constraint considered valid.
Note! If 'from' and 'to' are equal then it is only checked if the object can be put on the specified position. In that case this routine will never return CEL_MOVE_PARTIAL.
