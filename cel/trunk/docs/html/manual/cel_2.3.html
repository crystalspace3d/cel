2.3 External Libraries

Crystal Entity Layer currently has two external dependencies, the first of these of course being Crystal Space.

You may obtain Crystal Space from:
http://www.crystalspace3d.org/

The second dependency is Python. Python is a scripting language which is used by the Python Behaviour Layer. This is not required in order to build Crystal Entity Layer (it is an optional module), however if you wish to use Python in your own Behaviour Layer, or build the blpython example, then you will need to download Python from:
http://www.python.org/
